## Spring Deep Dive

스프링 프레임워크와 백엔드 개발의 핵심 개념을 실무 중심으로 학습하는 프로젝트입니다.

각 모듈은 독립적으로 학습할 수 있으며, 각 디렉토리의 `LEARN.md` 파일에서 상세한 학습 가이드를 확인할 수 있습니다.

---

각 모듈은 다음 구조로 구성되어 있습니다:

```
module-name/
├── LEARN.md              
│   ├── 언제 사용하는가?
│   ├── 핵심 개념
│   ├── 실습 시나리오
│   └── Best Practices
├── src/main/java/        
└── src/test/java/        
```

**학습 단계**:
1. `LEARN.md` 읽기 (개념 이해)
2. 코드 실행 및 디버깅 (동작 확인)
3. 테스트 코드 작성 (이해도 검증)
4. 실무 적용 사례 고민
